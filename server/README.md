## to setup the database be in the server-directory.

#DATABASE

Setup db:
$ cd server
$ node utils/setupDB.js

Data in db: 
$ cd server
$ node utils/seedDB.js

## to run the applikation local:

#FRONTEND:

$ cd client
$ npm run build
$ npm run dev

#BACKEND:

$ cd server
$ node/nodemon app.js
